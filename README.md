# TWRP Device Tree生成工具

- 这个工具怎么用？

-----

- 1.从原厂固件中提取boot.img或者recovery.img
  
-----

- 2.将这个项目fork到你自己的仓库

-----

- 3.将recovery.img或boot.img上传至 https://bashupload.com/

-----

- 4.点击actions － make twrp device － run workflow，将获取到的链接复制粘贴到IMG_URL

-----

 - 5、填写完成后点击 'Run workflow' 开始运行

-----
